### 5.2 Token 传递规范

<!-- 填写指引：定义前端发送 Token 的方式 -->

```http
# ✅ 正确：通过 Authorization Header 传递

# ❌ 错误：通过 URL 参数传递 Token（安全风险）

# ❌ 错误：通过 Cookie 传递（除非有 CSRF 保护）

```
