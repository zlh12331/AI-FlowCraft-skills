### 6.3 空值处理原则
1. **后端返回原则**：可选字段统一返回 `null`，不返回缺失字段。列表类型统一返回空数组 `[]`。
2. **前端展示原则**：所有可能为 null 的字段在展示前必须做空值判断，提供合理的降级展示。
3. **数据库存储原则**：可选字段允许 NULL，NOT NULL 字段必须有默认值。
